# 算法与数据结构 - TS 描述

## 前言

本库是以 **`TypeScript`** 为描述语言的算法与数据结构入门仓库。
主要面向想要学习或者想要查漏补缺的前端开发者。
本库可以作为算法与数据结构的字典查询，也可以作为入门资料学习。
对此感兴趣的欢迎 **star**
